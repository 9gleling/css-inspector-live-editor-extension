Privacy Policy

CSS Inspector does not collect, store, or transmit personal information.

Data Processing

All DOM inspection, accessibility analysis, typography analysis, and style editing are processed locally within the user's browser.

Permissions

This extension requires page access permissions in order to:

  • Inspect DOM elements
  • Analyze CSS styles
  • Provide live style editing
  • Analyze accessibility and typography

Storage

The extension may use Chrome local storage to save temporary UI preferences. No browsing history or personal information is stored.

Third-Party Services

CSS Inspector does not send data to external servers.
